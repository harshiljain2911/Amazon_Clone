# 🛒 Amazon Clone (MERN Stack)

A full-stack Amazon clone built using the MERN stack (MongoDB, Express, React, Node.js) with modern UI, authentication, cart system, and checkout flow.

---

## 🚀 Features

- 🔐 OTP-based Authentication
- 🛍️ Product Listing & Categories
- ❤️ Wishlist Functionality
- 🛒 Add to Cart with Quantity Control
- 💳 Checkout System (Mock Payment)
- 📧 Fake Order Confirmation Email (for demo)
- ☁️ Image Upload using Cloudinary
- 📦 MongoDB Database Integration
- ⚡ Responsive UI (Amazon-like design)

---

## 🧑‍💻 Tech Stack

### Frontend:
- React.js
- Redux Toolkit
- Tailwind CSS

### Backend:
- Node.js
- Express.js
- MongoDB (Mongoose)

### Services:
- Cloudinary (Image Upload)
- Nodemailer (Email System)
